This app is a static personal portfolio with no persistent database requirements. All features — including the contact form — are implemented via client-side React logic and a Vercel serverless function (`api/contact.ts`). The contact form uses Resend for email delivery and Upstash Redis for IP-based rate limiting (5/hour). No Supabase tables, storage buckets, or edge functions are used. The Supabase client is not included in the project. Backend functionality is fully decoupled into Vercel serverless functions owned by the Email Agent and API Integration Agent.
