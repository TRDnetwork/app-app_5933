This app is a static personal portfolio with no persistent data storage. All features are implemented client-side or via Vercel serverless functions. The contact form is processed by a serverless endpoint (`api/contact.ts`) that uses Resend for email delivery and Upstash Redis for rate limiting (5 submissions/hour/IP). No Supabase database tables, storage buckets, or edge functions are used. The frontend uses React, Vite, TypeScript, Tailwind CSS, and Framer Motion, deployed on Vercel.
