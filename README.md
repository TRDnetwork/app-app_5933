This app is a static personal portfolio with no persistent data storage. All features are implemented client-side or via Vercel serverless functions. The contact form is processed by `api/contact.ts`, which uses Resend to send emails and Upstash Redis for IP-based rate limiting (5/hr, fail-open). No Supabase database tables, storage buckets, or edge functions are used. The backend is fully decoupled and runs on Vercel's serverless infrastructure with input sanitization, honeypot spam protection, and Sentry logging.
